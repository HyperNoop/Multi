# Guide de Contribution

Merci de votre intérêt pour contribuer à Multi !

## Comment Contribuer

1. **Fork** le repository
2. **Clone** votre fork
3. Créez une **branche** pour votre fonctionnalité
4. **Testez** vos changements
5. **Committez** avec des messages clairs
6. **Push** vers votre fork
7. Ouvrez une **Pull Request**

## Standards de Code

- Code Python conforme à PEP 8
- Commentaires en anglais
- Documentation en français et anglais
- Tests pour les nouvelles fonctionnalités

## Ajouter une Langue

Pour ajouter une nouvelle langue:

1. Ajoutez les mots-clés dans `MULTI_DICTIONARY`
2. Testez la traduction
3. Ajoutez des exemples
4. Mettez à jour la documentation

## Questions

Ouvrez une issue pour toute question !
